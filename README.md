# Pandas Data Cleaning & Exploratory Data Analysis

## Overview
This project demonstrates data cleaning and exploratory data analysis (EDA) using Python and Pandas.
The goal is to prepare raw data for analysis and uncover meaningful patterns through structured exploration.

## Dataset
- File: world_population.csv
- Description: Country-level population data across multiple years

## Tools 
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Project Structure
- Data cleaning notebook
- Exploratory data analysis notebook
- CSV dataset

## Data Cleaning
- Handled missing and inconsistent values
- Renamed and standardized columns
- Converted data types for analysis
- Removed unnecessary rows and columns

## Exploratory Data Analysis (EDA)
- Summary statistics and distributions
- Country and year-based comparisons
- Trend analysis across time
- Identification of outliers and patterns

## Skills Demonstrated
- Data preparation and cleaning
- Exploratory data analysis
- Pandas-based data manipulation
- Analytical thinking with real-world data
